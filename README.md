## 🧰 Common Tasks

### Adding a Role 

1. Open discord-mcp.
2. Navigate to the User Management section.
3. Select "Add Role."
4. Type the role name and permissions.
5. Click "Save."

### Banning a User 

1. Go to Message Moderation.
2. Find the user by typing their name.
3. Click "Ban."
4. Confirm your choice.

### Setting Welcome Message 

1. Open Server Settings.
2. Select "Welcome Message."
3. Type the message you want new users to receive.
4. Click "Enable" and then "Save."
